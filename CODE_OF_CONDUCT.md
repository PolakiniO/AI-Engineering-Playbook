# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful to contributors and adopters from different backgrounds and experience levels.

Participants in this project are expected to engage in a constructive, professional, and inclusive way.

## Expected Behavior

- be respectful in discussion and review
- assume good intent while staying technically rigorous
- give actionable feedback
- focus criticism on ideas, code, and documentation rather than people
- help keep the project usable for a broad open-source audience

## Unacceptable Behavior

- harassment or personal attacks
- discriminatory or demeaning language
- sustained disruption of discussions or reviews
- publishing private information without permission
- hostile or exclusionary conduct

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, and other project communication channels.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening an issue or using the repository discussions feature. If a private reporting channel is required, maintainers will provide one in the future.
